## [1.0.0] - 2025-02-24

### Added

- Initial release as standalone open-source library.
- Components: Button, Input, Card, Badge, ModalPortal, ConfirmationDialog, Dropdown, Pagination, SearchInput, EmptyState, LoadingSpinner, Toast, ThemeToggle, ErrorBoundary, Form, DangerZone, TabNavigation, List, PageHeader, Alert, Progress.
- ThemeProvider and useThemeContext for dark/light mode.
- Full dark mode support via `.dark` class on document root.
- Storybook documentation.
- Unit tests (Vitest) and E2E tests (Playwright).
- CI (lint, typecheck, test, build, e2e) and release workflows for GitHub Actions.
- GitHub Pages deployment for Storybook docs.
